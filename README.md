## Описание проекта

Тестовое задание для стажировки Frontend разработчиков в Avito (осень 2025).  Проект выполнен на **React + JavaScript**.

Веб-приложение для модерации объявлений на платформе Авито. Это упрощённая версия внутренней системы, которую используют модераторы для проверки и управления объявлениями пользователей.

Задача — реализовать небольшое клиентское приложение согласно требованиям задания: работа с API, отображение данных, фильтры, навигация и базовая UI-логика.

## Функционал

- Получение данных с API.
- Отображение списка объявлений.
- Страница деталей: подробная информация по конкретному объявлению.
- Фильтрация и сортировка объявлений.
- Поиск.
- Роутинг между страницами.
- Обработка ошибок.

## Инструкция по запуску проекта

### Требования

- Node.js v20 или выше
- npm или yarn

### Установка и запуск

#### 1. Клонирование репозитория

```bash
git clone https://github.com/oryx3422/Avito-Frontend-assignment-autumn-2025
cd Avito-Frontend-assignment-autumn-2025
```

#### 2. Запуск сервера

```bash
# Переход в директорию сервера
cd server

# Установка зависимостей
npm install

# Запуск сервера
npm start

# Или запуск в режиме разработки с автоматической перезагрузкой
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:3001`

#### 3. Запуск клиентского приложения

Откройте новый терминал:

```bash
# Переход в директорию клиента
cd client

# Установка зависимостей
npm install

# Запуск приложения в режиме разработки
npm run dev
```

Клиентское приложение будет доступно по адресу: `http://localhost:3002`
