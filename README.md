# Система управления объявлениями для модерации

![scheme](<https://github.com/avito-tech/tech-internship/blob/main/Tech%20Internships/Frontend/Frontend-trainee-assignment-autumn-2025/schema%20(3).jpg>)

## Описание проекта

Веб-приложение для модерации объявлений на платформе Авито. Это упрощённая версия внутренней системы, которую используют модераторы для проверки и управления объявлениями пользователей.

### Основной функционал

1. **Список объявлений** (`/list`)

   - Отображение объявлений в виде карточек с информацией о товаре
   - Фильтрация по статусу, категории, диапазону цен
   - Поиск по названию
   - Сортировка по дате, цене, приоритету
   - Пагинация (10 объявлений на страницу)

2. **Детальный просмотр объявления** (`/item/:id`)

   - Полная информация об объявлении
   - Галерея изображений
   - История модерации
   - Панель действий модератора (одобрить/отклонить/вернуть на доработку)

3. **Статистика модератора** (`/stats`)
   - Общие метрики (проверено объявлений, процент одобренных/отклонённых)
   - Графики активности по дням
   - Круговая диаграмма распределения решений
   - График по категориям

## Инструкция по запуску проекта

### Требования

- Node.js v20 или выше
- npm или yarn

### Установка и запуск

#### 1. Клонирование репозитория

```bash
git clone https://github.com/oryx3422/Avito-Frontend-trainee-assignment-autumn-2025
cd Avito-Frontend-trainee-assignment-autumn-2025
```

#### 2. Запуск сервера

```bash
# Переход в директорию сервера
cd server

# Установка зависимостей
npm install

# Запуск сервера
npm start

# Или запуск в режиме разработки с автоматической перезагрузкой
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:3001`

#### 3. Запуск клиентского приложения

Откройте новый терминал:

```bash
# Переход в директорию клиента
cd client

# Установка зависимостей
npm install

# Запуск приложения в режиме разработки
npm run dev
```

Клиентское приложение будет доступно по адресу: `http://localhost:3002`
